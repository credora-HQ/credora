# Credora

Portable professional verification infrastructure.

Credora enables institutions to issue cryptographically verifiable credentials that holders can securely share with recruiters, employers, and platforms.

## Core Principles

- Verify claims, not people
- Proof over exposure
- Infrastructure, not judgment
- Privacy-preserving by design

## Features

- W3C Verifiable Credentials
- Live revocation system
- Controlled access profiles
- Issuer key registry
- Verification APIs
- PAN-anchored identity verification (India v1.0)

## Stack

- React / Next.js
- Node.js / FastAPI
- PostgreSQL / Redis
- Ed25519 signatures
- RFC 3161 timestamping

## Status

Currently in active development.
